Создадим шаблоны в accounts/templates/accounts/. Приведём примеры основных шаблонов с использованием crispy_forms и Bootstrap.

Дополнения
Crispy Forms: мы уже подключили crispy_forms и crispy_bootstrap5 в settings.py. Шаблоны теперь красиво рендерят формы.
Статичные файлы: Убедитесь, что у вас есть default_avatar.png в static/img/ для отображения, если аватар не загружен.
Сообщения (messages): Во всех шаблонах стоит добавить вывод сообщений. Например, в base.html перед {% block content %}:
html
Копировать код
{% if messages %}
<div class="container mt-2">
  {% for message in messages %}
    <div class="alert alert-{{ message.tags }}">
      {{ message }}
    </div>
  {% endfor %}
</div>
{% endif %}
