## О проекте

Это веб-приложение на Django для личных заметок и управления задачами (напоминаниями).  
Основные возможности:
- Регистрация и вход пользователей
- Создание, редактирование, удаление заметок с категориями и тегами
- Прикрепление файлов к заметкам
- Создание, редактирование, удаление задач с дедлайнами и статусами
- Поиск по тегам, категориям и ключевым словам
- Профиль пользователя с возможностью изменения аватара
- Простая статистика по количеству заметок и задач

## Структура проекта

_personal_notes_project/_ — корневая директория проекта Django;

_accounts/_ — управление пользователями (регистрация, вход, профиль);

_notes/_ — функционал заметок (создание, вывод списка, поиск, вложения);

_tasks/_ — функционал задач (напоминания с дедлайном, статусы);

_categories/, tags/_ — модели и представления для категорий и тегов;

_templates/, static/_ — шаблоны и статические файлы;

## Дополнительные функции

Поиск по тегам и категориям;

Изменение аватара пользователя;

Просмотр статистики (например, на главной странице или в профиле);

Админ-панель Django для управления всем контентом.
